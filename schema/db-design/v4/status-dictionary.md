# V4 状态字典（整数）

## 通用
- `0` 未指定
- `1` 启用/有效
- `-1` 删除/禁用

## listing_status
- `1` 草稿
- `2` 待审核
- `3` 已上架
- `4` 已下架
- `5` 已出租
- `-1` 删除

## audit_status
- `1` 待审核
- `2` 通过
- `3` 驳回
- `-1` 关闭

## appointment_status
- `1` 待确认
- `2` 已确认
- `3` 已完成
- `4` 已取消
- `-1` 已关闭

## requirement_status
- `1` 活跃
- `2` 已匹配
- `3` 已成交
- `4` 已暂停
- `-1` 已关闭
