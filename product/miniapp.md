# Miniapp

小程序是 C 端用户找房入口。

## 当前范围

- 微信登录 / 注册
- 用户 session 校验
- 房源浏览与搜索
- AI 找房入口

## 下一阶段

- 先接入小程序 HPD 展示层。
- 找房列表和详情读取 HPD 展示层数据，不直接读取 HMD。
- HPD 后端规划见 [../backend/miniapp-hpd.md](../backend/miniapp-hpd.md)。
