# Agents

本目录用于给不同端的开发 Agent 提供接手规则。新开任务时，先读 `global.md`，再读对应端文档。

## 通用

- [global.md](./global.md)：所有 Agent 都必须遵守的全局规则。
- [docs.md](./docs.md)：维护 shared-docs 时的规则。

## 后端

- [backend-go.md](./backend-go.md)：Go 后端开发 Agent 规则。
- [ai-chat-python.md](./ai-chat-python.md)：Python AI chat 后端 Agent 规则。

## 前端

- [frontend-miniapp.md](./frontend-miniapp.md)：小程序开发 Agent 规则。
- [frontend-publish.md](./frontend-publish.md)：发房系统前端 Agent 规则。
- [frontend-admin.md](./frontend-admin.md)：后台管理前端 Agent 规则。
